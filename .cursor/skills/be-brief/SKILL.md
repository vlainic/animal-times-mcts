# Be Brief

Keep responses brief but readable — short, not compact:
- Answer directly, no preamble
- Use headers or bullets when listing 3+ items — structure over walls of text
- One sentence per point; drop filler words
- Skip "you could also..." suggestions unless critical
- Code blocks are exempt — always complete and correct
